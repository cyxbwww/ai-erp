# AI 智能销售 ERP

## 目录
- `frontend`：Vue3 + TypeScript + Naive UI 前端
- `backend`：FastAPI + SQLAlchemy 后端

## 启动
### 前端
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### 后端
1. `cd backend`
2. `python -m venv .venv`
3. `.venv\Scripts\activate`
4. `pip install -r requirements.txt`
5. `uvicorn app.main:app --port 8000`
